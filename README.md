## 🎯 Overview

Welcome to my comprehensive portfolio documenting the completion of the **IBM Business Intelligence (BI) Analyst Professional Certificate**! This repository showcases hands-on projects, labs, and assignments covering the complete BI workflow—from data warehousing and SQL querying to advanced data visualization and dashboard creation.

## 🏆 Professional Certificate Details

- **Certificate**: IBM Business Intelligence (BI) Analyst Professional Certificate
- **Provider**: IBM via Coursera
- **Duration**: 9 comprehensive courses
- **Skills Acquired**: Business Intelligence, Data Visualization, Tableau, Excel, SQL, Statistical Analysis, Data Warehousing, Dashboard Design, ETL, Data Modeling

## 📚 Course Structure & Portfolio Contents

### 1. 📊 **Introduction to Business Intelligence (BI) Essentials**
- **Topics Covered**: BI concepts, analytics types (descriptive, diagnostic, predictive, prescriptive), BI ecosystem, data governance, stakeholder communication
- **Key Files**:
  - `Business Intelligence (BI) Essentials/Cheatsheet/` - Comprehensive course glossaries
  - `Interactive Glossary/` - Searchable BI terminology database
  - `Interactive Role Comparison Dashboard/` - Visual comparison of BI Analyst, Data Analyst, Data Engineer, and Data Scientist roles
  - `Final Project - Scenario/` - Real-world BI project scenario application

### 2. 🐘 **Introduction to Relational Databases (RDBMS)**
- **Topics Covered**: Database design, normalization, ER diagrams, MySQL, PostgreSQL, keys and constraints, views, materialized views
- **Key Projects**:
  - **Final Project - Database Design and Implementation**: Complete coffee shop database design with ERD, implementing tables, views, and materialized views
  - **Advanced Relational Model Concepts**: Python scripts demonstrating database connectivity and operations
  - **Views in PostgreSQL**: Creating and managing views for data abstraction and security
  - **Interactive RDBMS Glossary**: Comprehensive database terminology reference

### 3. 📈 **SQL: A Practical Introduction for Querying Databases**
- **Topics Covered**: SELECT statements, JOINs, subqueries, stored procedures, transactions, aggregate functions, string patterns, grouping, sorting
- **Key Projects**:
  - **Getting Started with SQL**: Comprehensive exercises with San Francisco film locations dataset
  - **Intermediate SQL**: Built-in functions, subqueries, multiple table operations
  - **Final Project - Advanced SQL Techniques**: Chicago crime, school, and socioeconomic data analysis with views, stored procedures, and transactions
  - **Interactive SQL Glossary**: Complete SQL reference with examples

### 4. 📉 **Statistical Analysis Fundamentals using Excel**
- **Topics Covered**: Descriptive statistics, measures of dispersion, probability distributions, normal distribution, simple/multiple linear regression, pivot charts
- **Key Projects**:
  - **Descriptive Statistics**: Mean, median, mode, variance, standard deviation calculations
  - **Measure of Dispersion**: Quartiles, percentiles, interquartile range analysis
  - **Probability Distribution**: Normal distribution calculations and market return simulations
  - **Simple & Multiple Linear Regression**: Sales prediction models for Zen Pharma
  - **Final Project**: Comprehensive supermarket sales transaction analysis with regression modeling
  - **Interactive Statistics Dashboard**: Complete statistical concepts reference

### 5. 📱 **Excel Basics for Data Analysis**
- **Topics Covered**: Excel functions, data cleaning, filtering, sorting, pivot tables, VLOOKUP, XLOOKUP, COUNTIFS, SUMIFS
- **Key Projects**:
  - **Advanced Excel Formulas Dashboard**: Interactive reference for Excel functions
  - **Data Cleaning Labs**: Flash fill, text functions, removing duplicates
  - **Pivot Table Analysis**: Indian startup funding and customer demographics analysis
  - **Final Project**: Montgomery Fleet Equipment Inventory - complete data cleaning and analysis workflow

### 6. 📊 **Data Visualization and Dashboards with Excel and Cognos**
- **Topics Covered**: Basic charts, advanced charts, pivot charts, dashboards, IBM Cognos Analytics, Google Looker Studio
- **Key Projects**:
  - **Complete Excel Mastery Lab**: Comprehensive Excel visualization workshop
  - **Excel Keyboard Shortcuts Interactive Lab**: Reference for Excel productivity
  - **Cognos Analytics Dashboards**: Interactive automotive industry dashboards
  - **Google Looker Studio**: Creating and configuring visualizations

### 7. 📈 **Getting Started with Tableau**
- **Topics Covered**: Tableau Public, data import, cleaning, visualizations, dashboards, stories, parameters, calculated fields
- **Key Projects**:
  - **Cleaning Data and Creating Visualizations**: Data preparation and chart creation
  - **Crafting Interactive Visualizations**: Interactive dashboard design
  - **Creating and Customizing Stories**: Narrative-driven data presentations
  - **Designing Interactive Dashboards**: Best practices in dashboard design
  - **Final Project - World Happiness Report**: Comprehensive analysis of global happiness drivers (2015-2023)

### 8. 📊 **Advanced Data Visualization with Tableau**
- **Topics Covered**: Hierarchies, dashboard actions, extensions, data modeling, calculated fields, advanced aggregations, dashboard optimization

## 🚀 **Getting Started**

### Prerequisites
- Tableau Public (free account)
- Microsoft Excel (or Excel Online)
- PostgreSQL and pgAdmin
- Python 3.7+ (for AI labs)
- Web browser for interactive dashboards

### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/IBM-Business-Intelligence-Portfolio.git
   ```
2. Navigate to specific project folders
3. For Tableau projects, download the `.twb` or `.twbx` files and open in Tableau Public
4. For Excel projects, open `.xlsx` files in Microsoft Excel
5. For SQL projects, import the dump files into PostgreSQL
6. For interactive HTML dashboards, open in any web browser
