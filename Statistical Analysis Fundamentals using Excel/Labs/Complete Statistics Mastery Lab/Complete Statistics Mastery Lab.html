📐 Variance Calculator

Calculate population variance and sample variance for your data.

Population Variance (σ²)
σ² = Σ(x - μ)² / N
Where:
• σ² = Population variance
• Σ = Summation
• x = Each value
• μ = Population mean
• N = Total number of values
Sample Variance (s²)
s² = Σ(x - x̄)² / (n - 1)
Where:
• s² = Sample variance
• Σ = Summation
• x = Each value
• x̄ = Sample mean
• n = Sample size
• (n-1) = Bessel's correction

Probability & Distributions

Understand probability concepts, normal distribution, z-scores, and hypothesis testing.

🎯 Z-Score Calculator

Calculate z-scores to standardize data and understand its position relative to the mean.

Z-Score Formula
z = (x - μ) / σ
Where:
• z = Standard score
• x = Individual value
• μ = Population mean
• σ = Population standard deviation
