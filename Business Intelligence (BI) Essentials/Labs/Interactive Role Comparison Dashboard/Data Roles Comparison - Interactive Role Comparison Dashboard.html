$72,106 Average Salary

Excel Tableau Power BI SQL Dashboard Creation Data Visualization

  • Design and maintain data models

DA

Data Analyst

$65,112 Average Salary

Formats, cleans, and adapts unstructured data into meaningful feedback. Uses descriptive analysis and static modeling approaches.

Python SQL Excel Statistical Analysis Hypothesis Testing Data Cleaning

Key Responsibilities:

  • Clean and prepare data for analysis
  • Perform exploratory data analysis
  • Create statistical models
  • Conduct hypothesis testing
  • Describe past and present data trends

$115,000 Average Salary

Designs, builds, and maintains infrastructure for data storage, processing, and analysis. Creates reliable data pipelines.

Scala/Go Hadoop Spark SSIS Data Pipelines Big Data

  • Create and optimize data pipelines
  • Maintain data storage systems
  • Ensure data reliability and availability

$98,789 Average Salary

Machine Learning Python/R Statistics TensorFlow Predictive Analytics Deep Learning

  • Conduct predictive analytics

  • Forecast future patterns
  • Develop AI algorithms
