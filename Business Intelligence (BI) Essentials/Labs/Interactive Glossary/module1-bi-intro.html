Module 1: Introduction to Business Intelligence

Core BI concepts, tools, roles, and foundational terminology. Covers BI platforms, analytics technologies, ETL processes, business strategies, and key performance indicators essential for BI professionals.

45+ Terms

Analytics technologies

A variety of data-driven methodologies and techniques to address queries, identify patterns, forecast outcomes, and automate decision-making processes.

Analytical strategies

A long-term decision for an organization to use data to take actions to satisfy organizational vision and mission.

Apache Airflow

An open-source workflow management platform for data engineering pipelines.

AWS data pipeline

A web service that helps you reliably process and move data between different AWS compute and storage services.

Azure Data Factory

Cloud-based ETL and data integration service that allows you to create data-driven workflows for orchestrating data movement.

Benchmarking

A process to measure your business's success compared to similar organizations to identify performance gaps.

Business analyst

A data professional who bridges the gap between business objectives and technical solutions.

Business intelligence

A combination of various technologies, tools, and methodologies that gather, analyze, and transform data into meaningful information.

Business intelligence analyst

A professional who analyzes, models, and visualizes data to help businesses make informed decisions.

Business intelligence architect

A professional who designs and implements solutions that help organizations in making decisions.

Business intelligence developer

An engineer who is in charge of developing, deploying, and maintaining BI interfaces.

Business intelligence initiative

The goal of organizations to improve their revenue, operational efficiency, and competitive edge with better decisions.

Business intelligence manager

A professional who uses data analysis to suggest and execute decisions for a company.

Business intelligence platform

A cluster of tools and applications that enables organizations to analyze data and get insights.

Business intelligence strategies

The blueprint that a company generates to use the data.

Business intelligence tools

Proprietary or open-source application software used to collect, process, analyze, and report large data.

Corporate performance management (CPM)

Software that integrates information to support operational plans and align KPIs for financial planning.

Customer relationship management (CRM)

Software that helps companies measure and control their lead generation and sales pipelines.

Data analysis

A process that involves cleaning, transforming, and modeling data to uncover useful information.

Data analyst

A data professional who gathers, analyzes, interprets, visualizes data, and weaves it into a story.

Data analytics

Focuses on extracting valuable information from data using various tools, techniques, and algorithms.

Data engineer

Responsible for building robust data architecture and easily accessible data warehouses.

Data governance analyst

Helps ensure data is accurate, consistent, and compliant with legal requirements.

Data integration

Combination of technical and business processes that combine data from disparate sources.

Data marts

Partitions in data warehouses to manage specific business functions or departments.

Data mining

The process of uncovering patterns and valuable information from large data sets.

Data modeling

The process of creating visual representations of information systems.

Data modeling tool

Software applications that help design, create, and manage data models.

Data repositories

Data sets identified to be mined for reporting and analysis (data archives or libraries).

Data science

Focuses on understanding data through analysis, exploration, and cleaning.

Data scientist

Develops algorithms, builds predictive models, and uncovers patterns in large data sets.

Data storytelling

Skill that involves analyzing data to communicate insights and influence decisions.

Data visualization

The graphical representation of information and data to understand trends and patterns.

Data warehouse

A warehouse that pulls together data from many sources into a single repository for analytics.

Descriptive analysis

Utilizing statistical techniques to explain or summarize data (descriptive statistics).

E-commerce

A platform for buying and selling products and services online.

Enterprise resource planning (ERP)

Software systems that enable businesses to automate and manage key business processes.

Extract, load, transform (ETL)

A process that extracts, loads, and transforms data from multiple sources to a data warehouse.

Flat files

Collection of data stored in a two-dimensional database with records and fields.

IBM InfoSphere DataStage

ETL tool part of IBM Information Platforms Solutions suite.

JavaScript

A high-level scripting language for web development.

Key performance indicators (KPI)

Performance measures for specific objectives that provide targets and insights.

Machine learning engineers

Programmers who construct algorithms and systems enabling machines to learn.

Online analytical processing (OLAP)

Software for conducting multidimensional analysis on large volumes of data.

Python

An agile, expressive, open-source programming language for data analysis.

Real-time BI

A tool that provides up-to-the-minute information for monitoring KPIs in real time.

Return on investment (ROI)

A performance measure used to evaluate the efficiency of an investment.

SQL server integration services (SSIS)

Enterprise data integration, transformation, and migration tool.

Statistical tool

A tool that converts, analyzes, interprets, and uses data in different forms.

Structured query language (SQL)

Computer language used to interact with relational databases.

Talend

ETL tool for data integration with solutions for data preparation and quality.
