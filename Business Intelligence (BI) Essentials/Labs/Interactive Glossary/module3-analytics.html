Module 3: BI Analytics Landscape

Analytics methodologies, statistical concepts, data governance, security, and business applications. Covers descriptive, predictive, and prescriptive analytics, KPIs, and data management.

40+ Terms

Anonymization

Modifying personal data so it is no longer possible to identify individuals.

Authentication

Confirming the identity of a user or application seeking access to data.

Authorization

Determining whether an authenticated user has the right to perform an operation.

Business intelligence

Technologies, tools, and methodologies that gather, analyze, and transform data.

Business intelligence tools

Proprietary or open-source software for collecting, processing, and reporting data.

Causal relationship

One event influences the occurrence of another event.

Central tendency

The statistical measure representing the central value of a distribution.

Corporate performance management (CPM)

Software integrating information to support operational plans and align KPIs.

Customer relationship management (CRM)

Software for measuring and controlling lead generation and sales pipelines.

Data analysis

Cleaning, transforming, and modeling data to uncover useful information.

Data analytics

Extracting valuable information from data using tools and techniques.

Data encryption

Translating data to unreadable format to protect sensitive information.

Data governance

Managing data to ensure availability, usability, integrity, and security.

Data integration

Combining data from disparate sources into meaningful information.

Data marts

Subsets of data warehouses for specific business functions or departments.

Data mining

Uncovering patterns and valuable information from large data sets.

Data modeling

Creating visual representations of information systems.

Data privacy

Protection of personal data from unauthorized access.

Data repositories

Data sets identified for reporting and analysis (data archives).

Data security

Protecting digital information from unauthorized access or theft.

Data storytelling

Analyzing data to communicate insights and influence decisions.

Data visualization

Graphical representation of information to understand trends and patterns.

Data warehouse

Storage architecture pulling data from many sources into a repository.

Descriptive analysis

Utilizing statistical techniques to explain or summarize data.

Descriptive analytics

Using historical and current data to spot trends and relationships.

Diagnostic analytics

Identifying why events occurred and discovering correlations.

Enterprise resource planning (ERP)

Software systems for automating and managing key business processes.

Extract, load, transform (ETL)

Process moving data from multiple sources to a data warehouse.

Flat files

Collection of data stored in a two-dimensional database.

Key performance indicators (KPI)

Performance measures for specific objectives providing targets and insights.

Machine learning

Using data and algorithms to imitate how humans learn.

Mean

The arithmetic average of a data set.

Median

The middle value of a sorted data set (not sensitive to outliers).

Metadata management

Processes and technologies for managing metadata effectively.

Metrics

Measures to track performance of specific business processes.

Mode

The most frequently occurring value in a data set.

Online analytical processing (OLAP)

Software for conducting multidimensional analysis on large data volumes.

Predictive analytics

Predicting future outcomes using historical data and techniques.

Prescriptive analytics

Recommending optimal courses of action based on analytics.

Probability theory

Mathematical framework for finding likelihood of specific outcomes.

Python

An agile, expressive programming language for data analysis.

Real-time BI

Processes and tools providing up-to-the-minute information and analysis.

Regression analysis

Statistical methods for finding relationships between variables.

Return on investment (ROI)

Performance measure for evaluating investment efficiency.

Self-service business intelligence (SSBI)

Allowing business users to analyze data without IT support.

Standard deviation

Measure of how tightly data clusters around the mean.

Statistical analysis

Collecting data and using statistics to identify trends and insights.

Statistical modeling

Using mathematical equations to analyze data and identify patterns.

Statistical tool

A tool that converts, analyzes, interprets, and uses data.

Structured query language (SQL)

Computer language for interacting with relational databases.

Time-series analysis

Analyzing data to uncover patterns in time-based data.
