COUNTIFS Function

Multiple Criteria Counting

Counts the number of cells that meet one or more criteria. An extension of COUNTIF that handles multiple conditions.

=COUNTIFS(criteria_range_1, criteria_1, [criteria_range_2, criteria_2], ...)
Multi-condition Counting Data Filtering Conditional Analysis Report Generation

Key Features:

  • Handles multiple criteria ranges
  • Supports operators like >, <, >=, <=, <>
  • Works with dates, numbers, and text
  • Case-insensitive comparisons
  • All ranges must be same size

Common Use Cases:

  • Count sales by region AND product
  • Track employee attendance by department AND month
  • Analyze survey responses by demographic AND answer
  • Count overdue tasks by project AND priority
S

SUMIFS Function

Multiple Criteria Summation

Adds all numbers in a range that meet multiple criteria. More powerful version of SUMIF for complex conditional summing.

=SUMIFS(sum_range, criteria_range_1, criteria_1, [criteria_range_2, criteria_2], ...)
Conditional Sum Financial Analysis Budget Tracking Sales Aggregation

Key Features:

  • Sum based on multiple conditions
  • Supports wildcards (* and ?)
  • Can use cell references for criteria
  • Handles date ranges efficiently
  • Ignores text values in sum_range

Common Use Cases:

  • Monthly sales totals by region AND product
  • Departmental budget tracking by quarter
  • Commission calculations with tiered criteria
  • Expense summing by category AND date range
X

XLOOKUP Function

Modern Lookup & Search

The ultimate lookup function that replaces VLOOKUP/HLOOKUP. Searches ranges and returns corresponding values with advanced options.

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])
Data Lookup Two-way Lookup Error Handling Flexible Search

Key Features:

  • Default exact match (unlike VLOOKUP)
  • Search left, right, up, or down
  • Built-in error handling
  • Wildcard matching support
  • Binary search for sorted data

Common Use Cases:

  • Product price lookups
  • Employee record retrieval
  • Two-dimensional table lookups
  • Last item in a list lookups
  • Approximate match searches

📝 Practical Examples

COUNTIFS Example

Count sales in West region over $500:

=COUNTIFS(B2:B100, "West", C2:C100, ">500")

Returns: Number of sales transactions meeting both criteria

SUMIFS Example

Sum Q1 sales for Product A in East region:

=SUMIFS(D2:D100, A2:A100, "Q1", B2:B100, "Product A", C2:C100, "East")

Returns: Total sales amount meeting all three criteria

XLOOKUP Example

Find price of "ProductX" with error handling:

=XLOOKUP("ProductX", Products, Prices, "Not Found", 0, 1)

Returns: Price or "Not Found" if product doesn't exist

💡 Tips & Best Practices

COUNTIFS Tips

  • Use & to combine operators: ">"&A1
  • Use "<>" for "not equal to"
  • All ranges must be same size
  • Works with dates as serial numbers

SUMIFS Tips

  • sum_range is always first argument
  • Criteria can be cell references
  • Use "" for empty criteria
  • Dates need DATE() function or serial numbers

XLOOKUP Tips

  • match_mode: 0=exact, -1=exact or smaller
  • search_mode: 1=first to last, -1=last to first
  • Use IFERROR(XLOOKUP(...),"") for compatibility
  • Perfect for two-way lookups with nesting
