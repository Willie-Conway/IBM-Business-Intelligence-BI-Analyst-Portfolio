SQL Aggregation

GROUP BY Extensions: CUBE, ROLLUP, GROUPING SETS

Shiny Auto Sales Example
-- Basic GROUP BY: Sales by auto class SELECT autoclassname, SUM(salesamount) AS total_sales FROM DNsales WHERE salestype = 'New' GROUP BY autoclassname ORDER BY total_sales DESC; -- Result: -- Compact $2,483,783.94 -- Full-size $2,269,450.15 -- Midsize $1,906,575.50 -- Subcompact $1,265,765.05
-- CUBE: Multi-dimensional aggregation with grand total SELECT salespersonname, autoclassname, SUM(salesamount) AS total_sales FROM DNsales WHERE salestype = 'New' GROUP BY CUBE(salespersonname, autoclassname); -- Creates all possible combinations: -- 1. By salesperson & auto class -- 2. By salesperson only -- 3. By auto class only -- 4. Grand total
-- ROLLUP: Hierarchical aggregation SELECT YEAR(saledate) AS sale_year, MONTH(saledate) AS sale_month, SUM(salesamount) AS total_sales FROM DNsales GROUP BY ROLLUP(YEAR(saledate), MONTH(saledate)); -- Creates hierarchical totals: -- 1. Year + Month level -- 2. Year level total -- 3. Grand total
-- GROUPING SETS: Custom aggregation sets SELECT salespersonname, autoclassname, SUM(salesamount) AS total_sales FROM DNsales WHERE salestype = 'New' GROUP BY GROUPING SETS ( (salespersonname), (autoclassname) ); -- Creates two independent aggregations: -- 1. Total sales by salesperson -- 2. Total sales by auto class -- NO grand total (unlike CUBE)
Function Purpose Output Use Case GROUP BY Basic aggregation One dimension summary Simple reporting CUBE All combinations + totals Complete cross-tabulation Multi-dimensional analysis ROLLUP Hierarchical totals Drill-down summaries Time hierarchies, categories GROUPING SETS Custom aggregations Multiple independent summaries Custom report requirements
